# selective-robustness-auditing



Auditing LLM-as-Judge Evaluators in Task-Oriented Dialogue: A Counterfactual Approach

LLM-as-judge evaluators are increasingly deployed to assess multi-turn task-oriented dialogue, yet their reliability under realistic input variation remains poorly characterized. We present a systematic audit that combines counterfactual perturbation with multi-level analysis, examining how judge behavior shifts across (i) structural and content-level perturbations of dialogue inputs, (ii) prompt and instruction conditions, and (iii) internal model states accessed through targeted interventions. By mapping sensitivity profiles across error types and analysis levels — behavioral, representational, and causal — we identify regions where automated evaluation is dependable and regions where it is not. We discuss implications for the deployment of LLM judges in high-stakes dialogue settings.
